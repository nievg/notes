Бележки за github

Създаване на репозитория

  1. Логваш се в https://github.com/
  2. Отваряш падашото меню + в горния десен ъгъл на сайта
  3. Избираш опцията New repository
  4. В полето Repository name пишеш името на репозиторията
  5. В полето Description пишеш описание на репозиторията
  6. Трябва да има отметка в радио бутона Public
  7. Слагаш отметка в Initialize this repository with a README
  8. Натискаш зеления бутон Create repository
