Бележки за github

Създаване на репозитория

  1. Логваш се в https://github.com/
  2. Отваряш падашото меню + в горния десен ъгъл на сайта
  3. Избираш опцията New repository
  4. В полето Repository name пишеш името на репозиторията
  5. В полето Description пишеш описание на репозиторията
  6. Трябва да има отметка в радио бутона Public
  7. Слагаш отметка в Initialize this repository with a README
  8. Натискаш зеления бутон Create repository

Клониране на репозитория

  1. Логваш се в github https://github.com/
  2. Цъкаш на Repositories
  3. Цъкаш върху името на репозиторията
  4. Натискаш зеления бутон Clone or download
  5. Цъкаш Use HTTPS (ако пише Use SSH пропускаш стъпката)
  6. Копираш генерирания линк https://github.com/u_name/name_of_repository.git
  7. Отваряш gitBash на твоя компютър
  8. Отиваш в папката, която ще съдържа работното копие на репозиторията
  9. В gitBash пишеш git clone и пействаш линка, който си копирал в стъпка 6

Това създава папка с името и съдържанието на репозиторията.
Изпълнява се само веднъж, когато започваш да работиш по проекта.
Може след като се пушнат всички промени да изтрие цялата папка, а когато е необходимо да се работи по проекта отново да се клонира от репозиторията
