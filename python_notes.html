списък[НАЧАЛО:КРАЙ:СТЪПКА] - срез на списък

НАЧАЛО - индекса на елемента от който започва среза. По подразбиране е 0

КРАЙ - краят на среза. Не се включва в среза. По подразбиране е дължината на списъка

СТЪПКА - стъпка (всеки втори елемент или всеки трети и т.н.). По подразбиране е 1

print(списък[:])   - отпечатва целия списък тъй като всички стойности са по подразбиране

print(списък[1:])  - отпечатва елементите от втория до последния включително

print(списък[:3])  - отпечатва елементите от началото до третия включително

print(списък[::2]) - отпечатва всеки втори елемент

print(списък[:-1]) - отпечатва всички елементи в обратен ред (по индекс)

Кортежи (tuple)

Кортежите са списъци, които не могат да се променят след като са създадени

Кортежите заемат по-малко ресурси от списъците

a = (43, 56, 45.23, 'd') - кортеж

b = [43, 56, 45.23, 'd'] - списък

print(a.__sizeof__()) - ще отпечата 56 (байта)

print(b.__sizeof__()) - ще отпечата 72 (байта)

a = () - празен кортеж

a = tuple('Hello') - а = ('H', 'e', 'l', 'l', 'o')

a = ('Hello') - това създава str променлива, а не кортеж

a = 43, 56, 45.23, 'd' - това също създава кортеж

Функциите за работа с кортежи са като тези за работа със списъци, без променящите списъците

Речници

По същество речниците са асоциативни масиви

d = {} - празен речник

d = {key : value, ключ : стойност} - синтаксис за създаване на речник

d = dict(key=value, ключ=стойност) - създава речник

d = dict([(key, value), (ключ, стойност)]) - създава речник

d = dict.fromkeys([key, ключ]) - речник със стойности NONE и ключове - стойности от списък

d = dict.fromkeys([key, ключ], 1) - речник със стойности 1 и ключове от списъкa

d = {'str': 1, 2: 3, 25.3: 4, False: 2, 'a': True}

- при този синтаксис ключовете могат да са str, int, float, boolean

print(d) - ще отпечата речника в конзолата така сякаш е написан като код
