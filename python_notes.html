Бележки за python

Инсталиране на python

  1. Отиваш на https://www.python.org
  2. Отваряш секцията Download
  3. Цъкаш на големия жълт бутон Download Python 3.7.0
  4. Стартираш python-3.7.0.exe
  5. Слагаш отметка пред Add python to PATH за конзолата
  6. Следваш помощника

В Linux python е вграден

Писане на python в конзолата

> python - влизаш в python

>>> print("Hello") - извежда в козолата Hello

>>> print(2+3) - извежда в козолата 5

>>> 2 ** 3 - извежда в козолата 8 (две на степен 3)

>>> 25 // 4 - извежда в козолата 6 т.е. броят на четворките съдържащи се в 25

>>> 25 % 4 - извежда в козолата 1 т.е. остатък при целочислено деление

>>> 3.14 - 1.1 - извежда в козолата 2.04

>>> 5 - 2.72 - извежда в козолата 2.28 т.е. каства имплицитно

>>> quit() - излизаш в козолата

>>> Ctrl + Z, Enter - излизаш в козолата

Стартиране на py файлове в конзолата

> python hello.py - ще изпълни hello.py

Променливи

Типът на променливата не се указва строго

test = "String" - стринг

test = 12 - цяло число

test = 12.34 - число с плаваща запетая

del test - унищожава променливата

Кастване на типове

test = int(input("Enter an integer")) - каства към int

test = float(num1) - каства към float

test = str(23) - каства към str

Ако подадеш на int() стринг, който не е цяло число ще върне грешка

Оператори за присвояване

test   = 5 - test получава стойност  5

test  += 5 - test получава стойност 10

test  -= 7 - test получава стойност  3

test  *= 6 - test получава стойност 18

test  /= 9 - test получава стойност  2.0

test **= 6 - test получава стойност 64.0

test //= 5 - test получава стойност 12.0

test  %= 7 - test получава стойност  5.0

test = "стр_" - test получава стойност стр_

test  *= 3 - test получава стойност стр_стр_стр_

Оператори за сравнение

== != > >= < <=

>>> a = "a"; b = "b"; a > b; - извежда в козолата True

>>> "d" < "c" - извежда в козолата False

>>> 5 > "c" - TypeError: '>' not supported between instances of 'int' and 'str'

Цикли

k = 0

while k < 10:

   този ред ще се изпълни, ако условието k < 10 е вярно

   k += 1 - този ред е задължителен :)

този ред е извън цикъла


continue - пропуска текущата итерация

break - прекратява изпълнението на цикъла


for i in "Hello, world!":

  print(i)


for i in "Hello, world!":

  if i == "r"

    break

  print(i)

else:

  print(i) - този ред ше се изпълни само, ако break не е настъпил

else:може да се използва и с цикъла while

Ако се използва continue вместо break реда след else винаги се изпълнява
